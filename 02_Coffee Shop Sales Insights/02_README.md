# Coffee Shop Sales Insights
![image](https://github.com/kaneda05/kaggle_datasets/blob/main/png/02_Coffee%20Shop%20Sales%20Insights.png)

このデータは日別で各コーヒーの売上から構成されている簡素なデータなので、
扱いやすい反面、顧客のデモグラがないのが惜しいと思った。

ただ、８種類あるコーヒーの中でも売れやすいコーヒーの時期（時間、曜日、月）などの分析が可能なので、
想定としては、どのタイミングでどのコーヒーが売れやすいのかを特定したり、
お店の売上を最大化するにはどうすれば良いのかといった戦略を立てるなどには使えそうだと思ったので、
[Coffee sales strategy planning](https://www.kaggle.com/code/masakane/coffee-sales-strategy-planning)と言う名前でnotebookを作成。


個人的にコーヒーは好きなので、暇だったら外部のサイトからコーヒーの種類などを調べて詳細を記載したりするかも。
